登录节点后台

用控制面安装时生成的管理员 Token 登录。登录后可以查看节点状态、新增节点、管理游戏路由规则、生成一键安装命令，并处理节点下线、排空或禁用。

管理员入口

Token 保存在本机浏览器，不会写入服务端。
服务器上可通过 /etc/xaccel-control-api/control-api.env 查看。
